Switch on and off code snippets on your site.

== Description ==

Installing this plugin you will be provided field tested code snippets to enhance your website. These code particles include lightweight security tweaks, cleaning functions and other useful elements. Please note however, that Useful Snippets is not a security plugin––most of securtiy enhancements should be made on server level.

Following snippets are available:
* Setting up security headers.
* Disabling XML-RPC.

Following cleaning enhancements are available:
* Removing Wordpress default SVGs.

== Frequently Asked Questions ==

= Will this plugin add any code to my HTACCESS file? =

No, this plugin will not touch any Wordpress files.
